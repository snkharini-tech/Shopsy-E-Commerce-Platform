# Shopsy Online E-Commerce Database Management System

## Project Overview

The **Shopsy Online E-Commerce Database Management System** is a relational database project designed to manage the operations of an online shopping platform. The system stores and manages customer details, seller information, products, categories, orders, payments, inventory, delivery status, and customer reviews in an organized and secure manner.

The project demonstrates how a Database Management System (DBMS) helps an e-commerce platform efficiently manage large amounts of data while ensuring accuracy, consistency, security, and fast data retrieval.

---

## Problem Statement

Managing customer, seller, product, order, payment, and inventory data manually is time-consuming and may result in errors, duplicate records, and poor data management. This project provides a structured database solution that improves efficiency, maintains data integrity, and supports smooth business operations.
